1) Install and configure a DNS Server\Service from scratch.
2) configure the server for a primary domain(i.e. hacker.local1)
**** please do not use a global domain, i.e. walla.co.il \ google \ etc ;)

3) configure the server with a dedicated zone file for the domain, i.e.: hacker.local1.zone


4) configure fully functional environment for the domain, i.e. www, ftp, mail, etc.(with relevant records, A, CNAME, MX)

5) smile and the world will smile back ;]
